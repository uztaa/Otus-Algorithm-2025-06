# Домашнее задание: Реализация и сравнение алгоритмов сортировки

## Цель задания
Реализовать алгоритмы сортировки (QuickSort, MergeSort, ExternalSort) и сравнить их эффективность.

---

## Уровень JUNIOR
- **QS1**: Реализовать алгоритм быстрой сортировки QuickSort. (+1 байт)
- **MS2**: Реализовать алгоритм сортировки слиянием MergeSort. (+1 байт)
- **Сравнение времени**:  
  Заполнить таблицу времени сортировки случайного массива размером 10², 10³, 10⁴, 10⁵, 10⁶ для каждого реализованного алгоритма.  
  Таймаут на сортировку одного массива — 2 минуты. (+2 байта)

---

## Уровень MIDDLE
- **Генератор файла**: Написать функцию `(N, T)` для генерации текстового файла из N строк, каждая строка — случайное число от 1 до T. (+1 байт)
- **ES1**: Реализовать внешний способ сортировки ExternalSort с созданием T разных файлов и последующим слиянием. (+1 байт)
- **ES2**: Реализовать внешний способ сортировки ExternalSort с двумя вспомогательными файлами. (+1 байт)
- **ES3**: Реализовать внешний способ сортировки ExternalSort с загрузкой в память блоков по 100 чисел, сортировкой блока и сохранением, затем применением алгоритма из ES2. (+1 байт)
- **Сравнение времени для файлов**:  
  Заполнить таблицу времени сортировки файлов с параметрами:  
  - N = 10², 10³, 10⁴, 10⁵, 10⁶  
  - T = 10, N  
  Всего 10 вариантов сочетания N и T. (+2 байта)

---

## Уровень SENIOR
- Комплексное тестирование алгоритмов сортировки (кроме внешней сортировки). (+5 байт)  
- Провести тесты на массивах с размерами:  
  1, 10, 100, 1,000, 10,000, 100,000, 1,000,000, 10,000,000 (по желанию)  
- Использовать различные типы данных:  
  - random — случайные числа  
  - digits — случайные цифры  
  - sorted — на 99% отсортированный массив  
  - revers — обратно отсортированный массив  
- Результаты записать в файл, где первая строка — размер массива, вторая строка — отсортированные элементы через пробел.

---

## Итог
- Указать, какие пункты выполнены, сколько байт набрано, сколько времени затрачено на каждый пункт.
- Приложить ссылку на код и таблицу с результатами.
- Указать язык программирования.

---

## Компетенции, развиваемые заданием
- Создание алгоритмов сортировки массивов  
- Оценка и сравнение сложности алгоритмов  
- Анализ алгоритмов по критериям  
- Использование рекурсии  
- Использование принципа «разделяй и властвуй»
