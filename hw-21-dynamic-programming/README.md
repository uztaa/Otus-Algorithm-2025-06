## 📋 Реализованные задачи

### JUNIOR часть (4 задачи × 1 байт = 4 байта)

| Задача | Описание | Алгоритм | Сложность |
|--------|---------|----------|-----------|
| **Task 1: Дроби** | Сложение дробей и сокращение результата | НОД (Евклид) | O(log n) |
| **Task 2: Ёлочка** | Максимальная сумма при спуске по треугольнику | DP на матрице | O(N²) |
| **Task 3: 5×8** | N-значные числа из {5,8} без трёх подряд | DP со состояниями | O(N) |
| **Task 4: Остров** | Количество связных компонент единиц в матрице | DFS обход | O(N²) |

### MIDDLE часть (4 этапа × 1-2 байта = 6 байт)

| Этап | Описание | Алгоритм | Сложность |
|------|---------|----------|-----------|
| **Task 5: Маленький сарай** | Максимальный прямоугольник из нулей (N,M≤30) | Перебор углов | O(N³M³) |
| **Task 6: Длина сарая** | Высоты свободных клеток в направлении "вверх" | DP таблица | O(N×M) |
| **Task 7: Ширина сарая** | Левая и правая граница для каждого элемента | Монотонный стек | O(N) |
| **Task 8: Большой сарай** | Максимальный прямоугольник в матрице | Гистограмма | O(N×M) |

### Итого: 10 байт ✅

## 🧪 Система тестирования

### Unit-тесты (8 файлов)

Для каждой задачи создан файл с полным набором тестов:

- **test_task1.cpp** — Тесты НОД и операций с дробями
  - Простые примеры
  - Граничные случаи (НОД=1, одинаковые дроби)
  - Сокращение результата

- **test_task2.cpp** — Тесты DP на треугольнике
  - Малые треугольники
  - Одна строка
  - Максимальные значения

- **test_task3.cpp** — Тесты DP со состояниями
  - N от 1 до 10
  - Проверка корректности считающей для трёх подряд

- **test_task4.cpp** — Тесты DFS
  - Изолированные острова
  - Связанные компоненты
  - Пустая матрица

- **test_task5.cpp** — Тесты перебора прямоугольников
  - Полностью свободная матрица
  - Одно препятствие
  - Несколько препятствий

- **test_task6.cpp** — Тесты DP высот
  - Без препятствий
  - С препятствиями
  - Граничные строки

- **test_task7.cpp** — Тесты монотонного стека
  - Возрастающий массив
  - Убывающий массив
  - Смешанный массив

- **test_task8.cpp** — Тесты полного решения
  - Объединение всех этапов
  - Различные конфигурации препятствий

### Примеры тестовых случаев

```cpp
// Task 1: Дроби
EXPECT_EQ(TaskOneFractions::execute("1/2+1/3"), "5/6");
EXPECT_EQ(TaskOneFractions::execute("1/4+1/4"), "1/2");
EXPECT_EQ(TaskOneFractions::execute("2/3+1/3"), "1/1");

// Task 2: Ёлочка
std::vector<std::vector<int>> tree = {{1}, {2, 3}, {4, 5, 6}};
EXPECT_EQ(TaskTwoGarland::execute(tree), 10);

// Task 3: 5×8
EXPECT_EQ(TaskTheFiveEight::execute(1), 2);   // "5", "8"
EXPECT_EQ(TaskTheFiveEight::execute(2), 4);   // "55", "58", "85", "88"
EXPECT_EQ(TaskTheFiveEight::execute(3), 6);   // без "555", "888"

// Task 4: Остров
std::vector<std::vector<int>> matrix = {{1, 0, 1}, {0, 1, 1}, {1, 0, 1}};
EXPECT_EQ(TaskFourIslands::execute(matrix), 3);

// Task 5-8: Сарай с разными конфигурациями
```
