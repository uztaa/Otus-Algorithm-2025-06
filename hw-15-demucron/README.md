# ДЗ-15: Алгоритмы на графах

Реализация алгоритмов топологической сортировки и анализа связности графов с использованием собственных структур данных из ДЗ-04 (arrayslib).

## Реализованные алгоритмы

### 1. Алгоритм Демукрона (основное задание)

**Назначение:** топологическая сортировка DAG с разбиением вершин на уровни.

**Особенность:** в отличие от алгоритма Кана, который выдаёт линейный порядок, Демукрон группирует вершины по уровням. Вершины одного уровня можно обрабатывать параллельно.

**Применение:** планирование задач, параллельная компиляция, анализ зависимостей.

```cpp
SpaceArray<int> adjList;
adjList.add(0, 1);  // ребро 0 → 1
adjList.add(0, 2);  // ребро 0 → 2
adjList.add(1, 3);
adjList.add(2, 3);

DemukronResult result = DemukronAlgorithm::execute(adjList, 4);
// Уровень 0: [0]
// Уровень 1: [1, 2]  ← можно обрабатывать параллельно
// Уровень 2: [3]
```

**Сложность:** O(V + E)

---

### 2. Алгоритм Тарьяна (дополнительное задание)

**Назначение:** поиск компонент сильной связности (SCC) в ориентированном графе.

**Компонента сильной связности** — максимальное множество вершин, в котором из любой вершины можно достичь любую другую по направленным рёбрам.

**Применение:** анализ социальных сетей, оптимизация компиляторов, поиск циклических зависимостей.

```cpp
SpaceArray<int> adjList;
adjList.add(0, 1);
adjList.add(1, 2);
adjList.add(2, 0);  // цикл 0 → 1 → 2 → 0
adjList.add(2, 3);

TarjanResult result = TarjanSCC::execute(adjList, 4);
// Компонента 0: {0, 1, 2}  ← образуют цикл
// Компонента 1: {3}
```

**Ключевая идея:** DFS с двумя массивами:
- `disc[v]` — время обнаружения вершины
- `low[v]` — минимальное время, достижимое из поддерева v

Вершина v — корень SCC, если `low[v] == disc[v]`.

**Сложность:** O(V + E)

---

### 3. Мосты и точки сочленения (дополнительное задание)

**Назначение:** поиск критических элементов в неориентированном графе.

- **Мост** — ребро, удаление которого увеличивает число компонент связности
- **Точка сочленения** — вершина, удаление которой увеличивает число компонент связности

**Применение:** анализ надёжности сетей, поиск уязвимых мест в инфраструктуре.

```cpp
SpaceArray<int> adjList;
// Неориентированный граф: рёбра добавляем в обе стороны
adjList.add(0, 1); adjList.add(1, 0);
adjList.add(1, 2); adjList.add(2, 1);
adjList.add(1, 3); adjList.add(3, 1);

BridgesAPResult result = BridgesAndAP::execute(adjList, 4);
// Мосты: (1,2), (1,3)
// Точки сочленения: {1}  ← удаление вершины 1 разобщит граф
```

**Критерии:**
- Ребро (u, v) — мост, если `low[v] > disc[u]`
- Вершина u — точка сочленения, если:
  - u — корень DFS с ≥2 детьми, или
  - существует ребёнок v с `low[v] >= disc[u]`

**Сложность:** O(V + E)

## Формат входных данных

Граф задаётся списком смежности через `SpaceArray<int>`:

```cpp
SpaceArray<int> adjList;

// Добавление ребра from → to:
adjList.add(from, to);

// Для неориентированного графа — добавляем в обе стороны:
adjList.add(u, v);
adjList.add(v, u);
```
